# SSH Penetration Testing Lab

## Overview
This project demonstrates SSH enumeration and password auditing using Kali Linux and Ubuntu Server.

## Tools Used
- Kali Linux
- Ubuntu Server
- Nmap
- Hydra
- VMware Workstation

## Skills Demonstrated
- Network scanning
- SSH enumeration
- Password auditing
- Ethical hacking fundamentals

## Disclaimer
This project was performed in a controlled lab environment strictly for educational purposes.
